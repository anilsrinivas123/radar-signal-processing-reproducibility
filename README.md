# Radar and Signal Processing — Reproducibility Studies

This repository contains self-reproduced implementations and analyses of research papers
in signal processing, spectral estimation, optimization, and related areas.

Each subdirectory corresponds to a single paper and contains:
- MATLAB code
- a technical report (PDF)
- a paper-specific README describing methodology and scope

## Contents
- paper_1: Subspace methods and Nyström-based DOA estimation
- paper_2: Sparse linear systems and convex optimization for Sudoku
- paper_3: FFT-based three-point frequency estimation

This repository is intended for educational and research purposes.
